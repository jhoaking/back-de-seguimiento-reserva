# seguimiento reserva

1. Clonar repositorio

2. ejecutar `pnpm i `

3. cambiar las variables de entorno `.env.template` a `.env`

4. levantar el servidor con

```
docker-compose up -d
```

5. Ejecutar el servidor
   `pnpm run start:dev`

6. Ejecutar seed para levantar los 'servicios'

`http://localhost:3000/seed/`
